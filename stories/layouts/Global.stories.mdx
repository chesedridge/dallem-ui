# GlobalStyle

마지막 업데이트일: 2022년 5월 3일 KST

## 개요

CSS 미디어 쿼리를 미리 규정해둔 Styled-Components 태그입니다.

## 사용

프로젝트의 최상단 컴포넌트(index.tsx 또는 App.tsx)에 삽입하여 사용합니다.

```
import { GlobalStyle } from "dallem-ui";

export default function Index() {
    ...
    return (
        <>
            <GlobalStyle />
            ...
        </>
    );
};
```

설정되어 있는 값은 아래와 같습니다.

```
font-family: Pretendard, -apple-system, BlinkMacSystemFont, system-ui,
      Roboto, "Helvetica Neue", "Segoe UI", "Apple SD Gothic Neo",
      "Noto Sans KR", "Malgun Gothic", sans-serif;

font-size: 10px;
```
