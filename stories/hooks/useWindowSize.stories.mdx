# useWindowSize

마지막 업데이트일: 2022년 1월 19일 KST

## 개요

useWindowSize는 현재 브라우저의 길이와 높이를 반환하는 Hooks입니다.

## 사용

브라우저의 길이 또는 높이를 구해야 하는 컴포넌트에 삽입하여 사용합니다. <br />
아래는 가로 길이를 불러오는 예시입니다.

```
import { useWindowSize } from "dallem-ui";

const MyComponent = () => {
    const { width } = useWindowSize();
    ...
};
```

오브젝트 값으로 리턴되며, 오브젝트 값은 아래와 같습니다.

```
{
    width: number,
    height: number
}
```
