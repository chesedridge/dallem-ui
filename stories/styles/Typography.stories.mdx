# Typography

마지막 업데이트일: 2022년 1월 20일 KST

## 개요

달램 프로덕트에 사용되는 폰트 크기를 모아 둔 Object입니다.

## 사용

달램 타이포그래피가 필요한 컴포넌트에 삽입하여 사용합니다. <br />
아래는 Title을 불러오는 예시입니다. _GlobalStyle이 적용되어 있어야 정상적으로 작동합니다._

```
import { typography } from "dallem-ui";
import styled from "styled-components";

const MyConponent = () => {
    const Title = styled.p`
        ${typography.title}
    `
    return <Title>타이틀 테스트</Title>
}
```

오브젝트 내에 존재하는 값은 다음과 같습니다.
