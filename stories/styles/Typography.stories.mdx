# Typography

마지막 업데이트일: 2022년 5월 3일 KST

## 개요

달램 프로덕트에 사용되는 폰트 값을 Styled-Components로 구성한 태그 모음입니다,

## 사용

달램 타이포그래피가 필요한 컴포넌트에 삽입하여 사용합니다. <br />
아래는 H1(Head)을 불러오는 예시입니다. _[GlobalStyle](?path=/story/layouts-globalstyle--page)이 적용되어 있어야 정상적으로 작동합니다._

```
import { Text } from "dallem-ui";
import styled from "styled-components";

const MyConponent = () => (
    <Text.H1>타이틀 테스트</Text.H1>
);
```

사용 가능한 속성 값은 아래와 같습니다.

| Name  | Type   | Description                                                                                 |
| :---- | ------ | ------------------------------------------------------------------------------------------- |
| color | string | _optional_ HEX Code or [HTML Color Name](https://www.w3schools.com/colors/colors_names.asp) |

예시

```
import { Text } from 'dallem-ui'

const MyComponent = () => (
    <Text.H1 color="green">속성 값 테스트</Text.H1>
);
```

<Text.H1 color="green">속성 값 테스트</Text.H1>
