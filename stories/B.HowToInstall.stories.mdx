# 달램 UI 라이브러리 설치하기

마지막 업데이트일: 2022년 1월 20일 KST

## 프로젝트에 라이브러리 추가하기

_호환성의 문제가 발생하지 않는 한, yarn berry를 사용하는 것을 권장합니다._

```
yarn add dallem-ui
// or
npm i dallem-ui
```

## 폰트 적용하기

달램 UI는 [Pretendard](https://cactus.tistory.com/306)를 사용합니다.<br />
각 프로젝트별 최상위 컴포넌트(대체로 index.ts)에 폰트를 우선적으로 적용시켜주시면 됩니다.
**다이나믹 서브셋**을 기본으로 사용합니다.

## Styled-Components 적용하기

달램 UI는 [Styled-Components](https://styled-components.com/)를 사용합니다.<br />
각 프로젝트에 라이브러리를 다운 받으시면 됩니다.

## 기초 세팅 끝!

필요한 컴포넌트 / 스타일이 있다면 해당 문서를 열람하셔서 사용 방법을 확인하세요!
